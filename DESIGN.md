# PTC Creative — Design Direction

## Three Homepage Concepts

### Concept A — Capability Stack (Layered Technical Canvas)
A full-viewport technical canvas with five parallax layers: Brand → Material → Production → Digital → Growth.
Mouse movement shifts depth, connects nodes, and reveals material compositions (wood grain, acrylic edge, metal cut-path, light emission, UI chrome).
Hero feels like a live CAM/CAD composition — precise, cold-warm industrial, highly demonstrative.

### Concept B — Impact Engine (Creative Technology OS)
The homepage behaves as a production operating system.
Asymmetric bento workstations for each capability, a central Project Lab “machine”, and an AI Project Workspace (side panel, not chat bubble).
Navigation reads like a control surface; services are interconnected modules, not brochure cards.
Strong product feel: filters, modules, brief generation, solution stacks.

### Concept C — Material → Signal (Sticky Transformation Narrative)
A long-form sticky storytelling page where an idea physically transforms through materials into digital growth.
Horizontal chapters for portfolio, industry morph environments, heavy editorial Swiss typography.
Most cinematic; less “tool”, more filmic craft.

## Selected: Concept B — Impact Engine (with A’s canvas hero)

**Why:** Best matches “Creative Technology Platform”. Combines industrial precision with interactive product utility (Project Lab + AI workspace). Avoids agency brochure clichés while remaining commercially clear for German SMEs. Hero borrows Concept A’s layered canvas for theatrical entry; body stays OS-modular.

## Visual Direction
- Industrial precision + Swiss editorial + digital craftsmanship
- Graphite dark / warm concrete light
- Single accent: Electric Orange `#FF4D00`
- Monospace for specs, materials, data
- Soft technical grids, cut-paths, material swatches — no stock photography

## Typography
- Display / UI: Geist Sans (geometric, VI/DE capable)
- Specs / data: Geist Mono
- Hierarchy: display 56–80 / h2 32–40 / body 15–16 / mono 12–13

## Layout
- 12-col asymmetric grids, bento rhythm, max-w 1440
- Section padding: 96–128 desktop / 56–72 mobile
- Radius: 2–6px (sharp craft), rarely >12px
- Accent used only for active state, CTA primary, signal marks
